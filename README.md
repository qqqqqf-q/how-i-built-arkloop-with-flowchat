# How I Built Arkloop

这个仓库记录了我在构建 [Arkloop](https://github.com/ArkloopAI/Arkloop) 过程中，与 AI 的真实对话记录。

Arkloop 是一个开源的 AI Agent 客户端，本地优先，简单优先。我一个人打磨了三个月，光语音输入的 prompt 量就超过了 50 万字。

这里不是教程，也不是方法论。只是把真实的开发过程摊开来给你看 — 哪些地方顺利，哪些地方反复，哪些地方我改了十遍才满意。

## 结构

每个功能一个文件，按时间线排列。每个文件只保留 **我的 prompt** 和 **AI 的输出**，中间的工具调用、思考过程等已移除。

> 内容持续更新中。
